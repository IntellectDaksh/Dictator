# Security

Dictator runs entirely local: audio is transcribed in memory and discarded,
text is typed via simulated keystrokes (clipboard is never touched), and the
only network call is to Ollama on `localhost`. No telemetry, no accounts, no
API keys, no data leaves the machine.

Found a real vulnerability (not a bug report)? Email
uideasofficial@gmail.com instead of opening a public issue.
